SearchTubes･

Feb 02, 2018 Redundant but fun! Tags: Javascript react

Yet another small React project, I make these to get more comfortable with the framework and Javascript. I made this one with the intent on consuming an API.

Things I learned with this project: * Better structure, separation of concerns, a single point of entry. That is to say I only need to fetch the API in one place and send out that data to all the children components. I have another React project that I’m going to refactor as I was calling the API on several different components, even at the time of doing it I knew it was wrong (DRY) but did it anyway.

  • Component life cycles. Granted I didn’t make use of them here much outside of componentWillMount(), I’ve become familiar with the others to some extent, at the very least I’m aware of their uses.

To install:

  • clone git repository
  • at the command line run yarn start
  • localhost:8080
